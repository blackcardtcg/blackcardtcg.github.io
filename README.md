# BLACKCARD Vending — เว็บแสดงสินค้าในตู้

เว็บไซต์สำหรับลูกค้า แสดงสินค้า ราคา และจำนวนคงเหลือของแต่ละช่องในตู้ BLACKCARD ทั้ง 3 สาขา (เซ็นทรัล นครปฐม / สยามดิสคัฟเวอรี่ / เดอะมอลล์ บางแค) โดยดึงข้อมูลอัตโนมัติจากระบบหลังบ้าน Worldwide Vending VMS ทุก 30 นาที

## โครงสร้างไฟล์

- `docs/index.html` — หน้าเว็บลูกค้า (ธีมดำ-ทอง BLACKCARD)
- `docs/data.json` — ข้อมูลสินค้า/สต๊อก (สร้างโดยสคริปต์ อย่าแก้เอง)
- `sync.js` — สคริปต์ดึงข้อมูลจาก VMS
- `.github/workflows/update.yml` — ตั้งเวลารันอัตโนมัติทุก 30 นาที

## วิธีติดตั้ง (ทำครั้งเดียว ประมาณ 10 นาที)

1. สมัคร/เข้าสู่ระบบ GitHub ที่ https://github.com แล้วกด **New repository**
   - ตั้งชื่อ เช่น `blackcard-vending`
   - เลือก **Public** (จำเป็นสำหรับ GitHub Pages ฟรี) แล้วกด Create

2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repository
   - วิธีง่ายสุด: หน้า repo กด **uploading an existing file** แล้วลากโฟลเดอร์ทั้งหมดลงไป
   - ต้องได้โครงสร้างตามด้านบน (โฟลเดอร์ `.github/workflows/` ต้องอยู่ครบ)

3. ใส่รหัสบัญชี VMS เป็น **Secret** (ปลอดภัย ไม่มีใครเห็น)
   - ไปที่ **Settings → Secrets and variables → Actions → New repository secret**
   - Name: `VMS_ACCOUNTS`
   - Secret: `ชื่อผู้ใช้1:รหัสผ่าน1,ชื่อผู้ใช้2:รหัสผ่าน2`
     (คั่นแต่ละบัญชีด้วยลูกน้ำ ไม่มีเว้นวรรค)

4. เปิด GitHub Pages
   - **Settings → Pages → Branch: main → Folder: /docs → Save**
   - รอ 1-2 นาที จะได้ลิงก์เว็บ เช่น `https://ชื่อคุณ.github.io/blackcard-vending/`

5. ทดสอบรันครั้งแรก
   - แท็บ **Actions → อัปเดตข้อมูลสินค้าในตู้ → Run workflow**
   - ถ้าขึ้นเครื่องหมายถูกเขียว = สำเร็จ เว็บจะแสดงข้อมูลล่าสุด

หลังจากนั้นระบบจะอัปเดตเองทุก 30 นาทีโดยอัตโนมัติ

## รันสคริปต์เองบนเครื่อง (ถ้าต้องการ)

ติดตั้ง Node.js 18 ขึ้นไป แล้วรัน:

```
set VMS_ACCOUNTS=user1:pass1,user2:pass2
node sync.js
```

ไฟล์ `docs/data.json` จะถูกอัปเดต จากนั้นเปิด `docs/index.html` ดูได้เลย

## หมายเหตุ

- ตู้ใหม่ที่เพิ่มเข้าบัญชี VMS จะปรากฏบนเว็บอัตโนมัติ ไม่ต้องแก้โค้ด
- ราคาดึงจาก Goods management ของแต่ละบัญชี ถ้าเพิ่มสินค้าใหม่ให้ใส่ราคาและรูปในระบบ VMS ตามปกติ
- แนะนำให้เปลี่ยนรหัสผ่าน VMS เป็นระยะ และอัปเดตค่าใน Secret ให้ตรงกัน
